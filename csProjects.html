EndFrame Tic Tac Toe


Client/Server Desktop Application




Semester long group project (team of 4) using Java and JavaFX.
Presented deliverables at the end of each sprint using agile approach.


Implements

  • client/server architecture
  • microservice architecture for the different servers hosting the program
  • sockets and network programming
  • multithreading
  • minimax algorithm to find the next optimal move for playing against the computer; worst case scenario is a tie
  • MySQL and JDBC

Features

  • server side UI displays: all accounts, online players, active games, and all games
  • register new accouts
  • login authentication
  • vs computer
  • vs Player
  • edit account informatio
  • spectating other player's games
  • history of all games played

Memories Android App


Memories is an Android App to store and organize your collection of photos into separate events or "memories".
The primary goal of this app is to provide users an easy and convenient way to look back at the special moments they've captured with their phone.


Features

  • Create new memories with title, location, and date
  • Create new posts with date, location, and description
  • Add photos from camera or album
  • Save images to phone under the file name "Memories"
  • Edit memory or post description
  • Delete post or memories
  • View all photos
  • Pinch to zoom view of post image

Android News App


This android app allows users to get the most recent news and articles with the ability to save them to read at a later time.
Users are able to search and browse from over 30,000 news sources with the use of an api from NewsAPI.org.


Features

  • Scroll and view top trending news
  • View articles in more detail from the hosted site

Serendipity Book Sellers


Bookstore Management System

Semester long project written in C++ implementing concepts and materials learned from CS 1C at Saddleback College.
Periodically changed object designs, data structures, search and sorting methods as new materials were learned throughout the course.


Features

  • Cashier Module: Purchase books and display total price to user
  • Reports Module: Calculates and reports wholesale value, retail value of the current inventory.
  • Invenroty Module: Provides interface to search, add, edit, and delete a book's information in inventory
