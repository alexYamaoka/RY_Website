Memories Android App

Memories is an Android App to store and organize your collection of photos into separate events or "memories". The primary goal of this app is to provide users an easy and convenient way to look back at the special moments they've captured with their phone.


Features


  • Create new memories with title, location, and date
  • Create new posts with date, location, and description
  • Add photos from camera or album
  • Save images to phone under the file name "Memories"
  • Edit memory or post description
  • Delete post or memories
  • View all photos
  • Pinch to zoom view of post image
https://github.com/alexYamaoka/Photo_Memories

Android News App

This android app allows users to get the most recent news and articles with the ability to save them to read at a later time. Users are able to search and browse from over 30,000 news sources with the using an api from NewsAPI.org.


Features


  • Scroll and view top trending news
  • View articles in more detail from the hosted site
  • Save an article
  • View all saved articles
